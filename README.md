#### 安装教程

1.  前端（如果不想编译代码，在`"./前端代码/build"`目录下有编译好的代码）
安装依赖
### `yarn init`
运行代码
### `yarn start`
编译代码
### `yarn build`
运行编译好的代码
### `cd ./前端代码`
### `serve -s build`

2.  后端
function graph配置后端代码和APIG触发器
创建桶
**都要开启跨域**
